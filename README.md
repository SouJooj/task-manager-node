# Task Manager - Fullstack

Sistema de gerenciamento de tarefas desenvolvido com Node.js, Express, MySQL e JavaScript puro no frontend.

Projeto criado com foco em aprendizado de backend, arquitetura MVC e integração com banco de dados relacional.

---

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- MySQL
- JavaScript (Frontend)
- HTML5
- CSS3
- Arquitetura MVC

---

## 📂 Estrutura do Projeto

task-manager-node/
│
├── public/ # Frontend (HTML, CSS, JS)
├── src/
│ ├── config/ # Configuração do banco
│ ├── controllers/ # Lógica da aplicação
│ ├── models/ # Comunicação com banco de dados
│ └── routes/ # Definição das rotas da API
│
├── server.js # Arquivo principal do servidor
├── package.json
└── README.md

---

## 🎯 Funcionalidades

- Criar tarefas
- Listar tarefas
- Atualizar status (concluída / pendente)
- Deletar tarefas
- Contador de tarefas
- Interface Dark Mode moderna

---

## 🧠 Conceitos aplicados

- API REST
- CRUD completo
- Arquitetura MVC
- Integração Backend ↔ Banco de Dados
- Consumo de API com fetch()
- Organização de código em camadas
